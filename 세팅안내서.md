# 깃허브 액션으로 카카오 선물하기 랭킹 매시간 자동 수집하기

내 컴퓨터를 켜둘 필요 없이, **깃허브(GitHub)가 매시간 대신 랭킹을 긁어서 저장**해 줍니다.
설치할 프로그램 없고, 전부 무료이며, 웹브라우저에서 클릭만 하면 됩니다.
쌓인 데이터는 나중에 언제든 검색·확인할 수 있어요.

전체 소요: 처음 세팅 10~15분. 그 뒤론 아무것도 안 해도 됩니다.

---

## STEP 1. 깃허브 계정 만들기 (무료)

1. https://github.com/signup 접속
2. 이메일 → 비밀번호 → 사용자 이름(영문, 예: `feelikewave`) 입력하고 가입
3. 요금제는 **Free(무료)** 선택

---

## STEP 2. 저장소(Repository) 만들기

1. 로그인 후 오른쪽 위 **＋ → New repository** 클릭
2. **Repository name**: `kakao-gift-tracker` (아무 이름이나 OK)
3. 공개 범위:
   - **Public(공개)** 권장 → 자동 수집이 **완전 무료·무제한**입니다.
     (저장되는 건 공개 랭킹 데이터뿐이라 민감정보 없음)
   - Private(비공개)로 하고 싶으면 가능하지만, 무료 사용 시간(월 2000분) 안에 쓰려면
     수집 주기를 2시간에 한 번으로 줄이는 걸 권장 → 필요하면 조유진에게 말해주세요.
4. 나머지는 그대로 두고 **Create repository** 클릭

---

## STEP 3. 파일 올리기

1. 방금 만든 저장소 화면에서 **Add file → Upload files** 클릭
2. 압축을 푼 `kakao_gift_tracker` 폴더 **안의 파일/폴더를 전부** 드래그해서 올립니다.
   - `collector.py`, `requirements.txt`, `세팅안내서.md`
   - `.github` 폴더 (그 안에 `workflows/collect.yml` 이 들어 있어야 함) ← **중요**
   - `data`, `debug` 폴더
3. 아래 **Commit changes** 초록 버튼 클릭

> `.github` 폴더가 안 올라가면 자동 실행이 안 됩니다. 폴더째 드래그하면 됩니다.

---

## STEP 4. 봇에게 저장 권한 주기 (한 번만, 필수)

수집한 데이터를 저장소에 다시 써야 하므로 권한을 켭니다.

1. 저장소 상단 **Settings** 탭
2. 왼쪽 메뉴 **Actions → General**
3. 맨 아래 **Workflow permissions** 에서
   **"Read and write permissions"** 선택 → **Save**

---

## STEP 5. 처음 딱 한 번 — 보정용 실행

1. 저장소 상단 **Actions** 탭 클릭
2. (처음이면 "I understand my workflows, enable them" 버튼이 보이면 눌러 활성화)
3. 왼쪽에서 **카카오 선물하기 랭킹 수집** 클릭
4. 오른쪽 **Run workflow** 버튼 → **debug 체크박스 켜기** → **Run workflow**
5. 1~2분 뒤 초록 체크(✅)가 뜨면 완료

그다음 저장소의 **`debug` 폴더**를 열면 `.json` 파일들이 생겨 있습니다.
**그 파일 하나를 열어서 내용을 복사해 조유진(Claude)에게 보내주세요.**
→ 이걸로 "위시 수 / 카테고리"가 카카오 데이터의 어느 값인지 정확히 맞춰 마무리합니다.

**같이 보내줄 것 하나 더:** 카카오 선물하기에서 **① 건강 전체** 탭과
**② 이너뷰티/다이어트** 탭을 각각 눌렀을 때 주소창에 뜨는 **URL 2개**도 복사해서 보내주세요.

---

## STEP 6. (보정 후) 자동 수집 시작

조유진이 보정을 마쳐서 파일을 다시 주면, 그 파일만 STEP 3처럼 덮어쓰기로 올리면 끝.
그 순간부터 **매시간 자동으로** 랭킹이 `data` 폴더에 쌓입니다.

- 최신 표: `data/latest_건강_전체.csv`, `data/latest_이너뷰티_다이어트.csv` (클릭하면 바로 보임)
- 전체 히스토리: `data/ranking.sqlite` (시간대별 그래프는 이걸로 만듭니다)

---

## 알아두면 좋은 것

- **정확히 매 정각은 아닐 수 있어요.** 깃허브가 바쁘면 몇 분~십몇 분 밀릴 수 있지만, 시간당 1회는 지켜집니다.
- **저장소가 60일간 아무 변화 없으면** 깃허브가 자동 실행을 잠글 수 있어요. 매시간 데이터가
  커밋되니 보통은 계속 살아있지만, 혹시 멈추면 Actions 탭에서 한 번 다시 켜주면 됩니다.
- 이건 **개인 참고용**입니다. 수집 데이터를 외부에 서비스로 공개·판매하는 것은 카카오 약관과
  부딪힐 수 있으니 조용히 참고 용도로만 쓰세요.
- 나중에 "이 데이터를 온글림처럼 예쁜 화면으로" 원하면, 같은 저장소에 대시보드도 무료로 붙일 수 있어요.
